## **File Transfer**
**All files will be deleted 30 days after upload**
**All files uploaded are not considered private and must not include and unapproved information**
To delete a uploaded file either select a different retention duration during upload or click the red fire extinguisher at the top left within 12 hours of upload to remove the file.
### Direct Link File Transfer
Just upload your file then copy the link to share with others.
Only users has the link will have access to the file.
[http://rossmgmt.mrts.unt.edu/FileTransfer/](http://rossmgmt.mrts.unt.edu/FileTransfer/)
### Public Access File Transfer
